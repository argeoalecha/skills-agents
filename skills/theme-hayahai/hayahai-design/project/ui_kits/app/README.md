# App — UI kit

A workspace/dashboard shell using the Hayah brand language. Demonstrates a typical product surface — sidebar nav, search topbar, and three click-through screens.

**Components**
- `Shell.jsx` — `AppShell`, `Sidebar`, `Topbar`
- `Icon.jsx` — small inline-SVG icon set (Lucide-style)
- `Screens.jsx` — `HomeScreen` (project grid), `ChatScreen` (AI chat), `SettingsScreen`

**Click-through:** Click a project on Home → opens Chat. Use sidebar to navigate between Home / Conversations / Settings.
