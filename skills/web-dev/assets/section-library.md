# Section Library

All sections available in the web-dev skill. Default set is included unless the user opts out.

## Core sections (always included)

| Section | Component | Key elements |
|---|---|---|
| Nav | `<Nav>` | Logo badge, nav links (smooth scroll), Auth pill, CTA button. Blur + border on scroll. |
| Hero | `<Hero>` | Dot-grid bg, gradient orbs, H1 tagline, sub-headline, 2 CTAs, stat card stack on right |
| Footer | `<Footer>` | Brand + subtagline, sitemap column, social links, stack badges, admin link |

## Optional sections (ask if they apply)

| Section | Component | When to include |
|---|---|---|
| Services | `<Services>` | Has distinct service lines / offerings |
| Work | `<Work>` | Has case studies or portfolio to show |
| About | `<About>` | Company story matters (older brand, niche credibility) |
| Team | `<Team>` | Team is a trust signal (consulting, law, medical) |
| Testimonials | `<Testimonials>` | Social proof is a conversion lever |
| Careers | `<Careers>` | Actively hiring or wants a talent pipeline |
| Contact | `<Contact>` | Lead capture is the primary CTA |

## Hero layout variants

| Variant | Layout | When |
|---|---|---|
| **Two-column** (default) | Headline left, stat stack right | Works for most B2B sites |
| **Centered** | Full-width centered H1 + CTA | Consumer, bold brand, editorial |
| **Split-image** | Text left, image/graphic right | When a hero visual exists |
| **Bento hero** | Grid of cards below headline | AI/SaaS showing product previews |

## Services grid variants

| Variant | Layout |
|---|---|
| **3-col grid** (default) | 3 columns, hover tint |
| **2-col grid** | Wider cards with more description |
| **Accordion list** | Vertical expand/collapse |

## Work / case studies variants

| Variant | Layout |
|---|---|
| **Numbered list** (default) | Left number, middle text, right metric |
| **Card grid** | 3-col cards with label badge |
| **Timeline** | Left rail with dots, entries on right |

## Contact variants

| Variant | Layout |
|---|---|
| **Two-column** (default) | Info + map left, form right |
| **Centered form** | Full-width form, info below |
| **Minimal** | Just email + phone, no form |

## Additional specialty sections (generate on request)

| Section | Description |
|---|---|
| **Pricing** | 3-tier card with feature lists and CTA |
| **FAQ** | Accordion with common questions |
| **Blog preview** | 3-card grid linking to blog posts |
| **Partners / logos** | Scrolling or static logo grid |
| **Newsletter** | Email capture → Supabase `subscribers` |
| **Process / how it works** | Numbered steps with icons |
| **Awards / certifications** | Badge/logo grid with labels |
| **Location map** | Embedded map + address card |
| **CTA banner** | Full-width accent band with CTA |
| **Video hero** | Background video or embedded player |
