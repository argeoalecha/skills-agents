# Landing — UI kit

A theme-switchable marketing site shell. Use the dropdown in the top nav to flip between Classic, Coral, Editorial, Bento, and Midnight.

**Components**
- `Nav.jsx` — sticky, glass-blurred nav with theme picker
- `Hero.jsx` — adapts headline, alignment, and bg per theme
- `Features.jsx` — bento grid (Bento theme uses asymmetric layout)
- `Sections.jsx` — `Metrics`, `CTA`, `Footer`

The kit demonstrates the full landing-page rhythm: nav → hero → features → metrics → CTA → footer.
